# 3D Holographic Speaker with Gesture Interaction

이 프로젝트는 라즈베리파이를 활용한 **3D 홀로그램 스피커** 시스템의 프로토타입입니다.  
스피커 음악 재생 시 앨범 표지가 3D 홀로그램으로 공중에 입체 투사되며, 사용자의 손가락 제스처(스와이프 및 탭)를 감지하여 앨범 회전 속도 조절, 탐색바 탐색, 볼륨 조정, 재생 일시정지 등을 수행할 수 있도록 설계되었습니다.

---

## 📂 프로젝트 구조

* **`app.py`**  
  Flask 및 Socket.IO 서버로, 3D 홀로그램 프론트엔드 서빙 및 외부 센서 제스처 신호를 실시간 중계합니다.
* **`simulated_gpio.py`**  
  (필요시) 하드웨어 핀 제어를 위한 가상 GPIO 모듈
* **`gesture_detector.py`**  
  라즈베리파이 센서/카메라 모듈을 모사하는 터미널 키보드 제스처 송신기입니다. 맥북 환경에서 카메라나 센서 하드웨어 없이 제스처 조작을 완벽하게 테스트할 수 있습니다.
* **`templates/index.html`**  
  Three.js WebGL 3D 렌더링 영역 및 글래스모피즘 오디오 설정 대시보드
* **`templates/control.html`**  
  모바일/원격 접속에 최적화된 무선 컨트롤러 페이지. 재생 싱크 및 캘리브레이션 원격 변경을 실시간 동기화합니다.
* **`static/js/hologram.js`**  
  - **4면 뷰포트 투사(Pyramid Hologram Layout)**: 단일 WebGL 컨텍스트 내에서 4개의 카메라 좌표계를 구성하여 상/하/좌/우 대칭 반사 이미지를 실시간 드로잉합니다.
  - **관성 물리 연산**: 스와이프 제스처 종료 후 속도에 맞추어 3D 모델이 관성을 가지고 회전하다가 서서히 멈춥니다.
  - **Web Audio API**: 음원의 베이스 및 주파수를 실시간 분석하여 3D 앨범 아트의 펄스 및 파티클 링의 움직임을 제어합니다.
  - **MediaPipe Hands**: 맥북 웹캠을 통해 브라우저 자체적으로 손가락 위치를 추적하여 마우스 없이 제어할 수 있는 브라우저 제스처 모듈이 장착되어 있습니다.
* **`static/css/style.css`**  
  홀로그램 투사를 위한 다크 백드롭 및 반응형 반사 레이아웃

## 🚀 로컬 환경 실행 및 테스트 방법 (macOS)

### 1. Flask 서버 실행
이전에 구성했던 가상환경(venv)을 공유해 그대로 실행할 수 있습니다.

```bash
# 가상환경 활성화 및 서버 기동
source ../raspberry-pi-simulator/venv/bin/activate
python3 app.py
```
* 서버가 정상 구동되면 브라우저에서 **[http://localhost:5001](http://localhost:5001)**에 접속합니다.

### 2. 제스처 조작 테스트 방법 (3가지 방식 지원)

#### 방식 A: 마우스 및 터치 드래그 (시뮬레이션)
* 화면 중앙 부근을 마우스로 빠르게 쓸어 넘기면 3D 앨범 카드가 관성에 의해 물리 회전합니다.
* 화면의 임의 영역을 **더블 클릭**하면 음악이 **재생 / 일시정지**됩니다.
* 화면 최상단 영역(Top 15%)을 클릭하면 재생 프로그레스바가 탐색됩니다.

#### 방식 B: 맥북 웹캠 손가락 트래킹 (MediaPipe)
* 대시보드 내 **"웹캠 손가락 인식 켜기 (Local Gesture)"** 체크박스를 활성화합니다.
* 카메라 권한 허용 후 카메라 앞에서 집게 손가락을 보여줍니다.
  * **중앙 영역**: 손가락을 대고 쓸면 3D 홀로그램이 회전합니다.
  * **최상단 영역**: 손가락을 좌우로 가리키면 곡 재생 위치가 움직입니다.
  * **최하단 영역**: 손가락을 왼쪽/오른쪽으로 빠르게 튕기면 이전/다음 곡으로 전환됩니다.

#### 방식 C: 외부 터미널 제스처 입력 (라즈베리파이 센서 모킹)
새로운 터미널 창을 열고 아래 명령어를 통해 키보드 조작 송신기를 실행합니다.

```bash
source ../raspberry-pi-simulator/venv/bin/activate
python3 gesture_detector.py
```
* `L` / `R` 키를 누르면 다음/이전 곡이 넘어가고, `Space` 키는 일시정지, `S` 키는 홀로그램에 물리 회전력을 가합니다.

## 🔌 실제 라즈베리파이로 포팅 시 하드웨어 연결 및 실행 가이드

### 0. 의존성 패키지 설치 및 실행 (라즈베리파이 4/5 기준)
라즈베리파이 터미널(또는 SSH)에서 아래 명령어를 수행하여 필요한 패키지를 설치하고 서버를 실행합니다:

```bash
# 1. SSH 접속
ssh mingyu@mingyu.local

# 2. 프로젝트 디렉토리 이동
cd ~/hologram-web

# 3. 시스템 패키지 설치
sudo apt update
sudo apt install -y python3-pip python3-venv python3-dev libgl1 libglib2.0-0

# 4. 가상환경 생성 및 활성화
python3 -m venv .venv
source .venv/bin/activate

# 5. pip 최신화 및 Python 패키지 설치
pip install --upgrade pip
pip install flask flask-socketio

# 6. Flask 서버 가동
python3 app.py
```

### 1. 디스플레이 & 피라미드 아크릴
* 라즈베리파이의 HDMI 포트에 디스플레이를 연결하고 크롬 브라우저를 전체 화면(Kiosk 모드)으로 기동시킵니다.
* 디스플레이 중앙의 교차선 가이드에 맞춰 아크릴 피라미드 반사판을 올려놓으면 물리 홀로그램 조형이 완성됩니다.

### 2. 카메라 모듈 연동 (OpenCV + MediaPipe)
* `gesture_detector.py` 내부의 `run_physical_camera_detector` 예시 주석을 참고하여 카메라 캡처 루프를 구성합니다.
* 감지된 좌표 및 핀치 이벤트를 동일하게 WebSocket `sio.emit('hardware_gesture', ...)`로 전송하면 수정 없이 완벽히 구동됩니다.

### 3. 무선 원격 제어 (Pure Viewer & Remote Control)
실제 라즈베리파이에 장착된 투사용 디스플레이에는 오직 홀로그램 메쉬만 깔끔하게 노출하고, 폰이나 다른 PC로 편리하게 무선 제어할 수 있습니다.
* **본체 투사 브라우저 실행**: `http://localhost:5001/` 주소로 브라우저를 띄웁니다. 메인 페이지에는 조작 패널이나 디버깅 레이아웃 마크업 자체가 완전히 배제되어 있어, 100% 깔끔한 투사용 3D 화면만 나타납니다.
* **무선 원격 제어 페이지 접속**: 스마트폰이나 태블릿 등 외부 장치 브라우저로 `http://<라즈베리파이-IP>:5001/control` (또는 맥북의 경우 `http://mingyu.local:5001/control`) 에 접속합니다.
  * 본체에서 돌아가는 실시간 곡명, 가수명, 앨범아트 회전 싱크 및 오디오 진행 시간(SeekBar) 정보가 매초 자동 갱신됩니다.
  * 리모컨에서 원격 재생/일시정지/탐색/곡넘김 조작을 수행하고, 감도 슬라이더 및 제미나이 텍스트 프롬프트를 원격 무선 전송하여 본체 동작을 편하게 테스트할 수 있습니다.
